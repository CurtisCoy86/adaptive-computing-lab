# Shellwalker Notes for Adaptive Computing Lab

Shellwalker is the first testbed for Adaptive Computing Lab.

## Current Role

Shellwalker is a Python terminal app that began as a survival RPG and evolved into a neurodivergent life assistant.

## Existing Features

- Medication tracking
- Task tracking
- Bill tracking
- Check-ins
- Check-in trends
- Startup daily brief
- Edit/remove commands
- Tests
- Git history

## Why Shellwalker Works as a Lab

Shellwalker is:

- Small enough to understand
- Personal enough to matter
- Practical enough to test
- Flexible enough to evolve
- Already connected to real cognitive-load problems

## Near-Term Direction

The first adaptive features should focus on:

- Better daily brief behavior
- One recommended next action
- Overwhelm detection
- Check-in pattern summaries
- Gentle language adaptation

## Rule

Do not turn Shellwalker into a giant system too quickly.

Use it as an experiment bench.

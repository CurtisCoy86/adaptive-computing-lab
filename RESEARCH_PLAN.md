# Research Plan v0.1

## Main Research Question

How can small software systems observe user behavior, reduce cognitive load, adapt to individual needs, and become easier to use over time?

## Current Testbed

Shellwalker is the first Adaptive Computing Lab testbed.

## Initial Research Questions

### 1. Can software recognize overwhelm?

Can Shellwalker detect signs that the user may be overloaded, such as too many overdue tasks, missed medications, negative check-ins, or repeated incomplete goals?

### 2. Can software reduce the next action?

Instead of showing everything, can the system choose one useful next step?

### 3. Can software adapt its language?

Can the system learn whether the user responds better to direct, gentle, structured, minimal, or encouraging language?

### 4. Can software identify useful patterns?

Can check-ins and logs reveal trends that are useful without becoming invasive or judgmental?

### 5. Can software become easier through use?

Can Shellwalker require less manual effort over time by remembering common patterns, recurring tasks, and preferred routines?

### 6. Can agents reliably build small adaptive features?

Can a builder/tester/reviewer/reporter workflow produce better experimental code than a single-pass coding workflow?
